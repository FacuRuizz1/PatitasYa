# 🐾 PatitasYa
Plataforma web para reportar mascotas perdidas, encontradas o en adopción 🐶🐱
Conectando personas para ayudar a que más mascotas vuelvan a casa ❤️

# 🚀 Descripción
PatitasYa es una aplicación Full Stack que permite a los usuarios publicar reportes de mascotas, subir imágenes y gestionar sus publicaciones de forma segura.
